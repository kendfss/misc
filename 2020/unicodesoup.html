Components of The Unicode® Standard
Version 13.0.0

This page lists the components of Version 13.0.0 of the Unicode Standard. The version numbering and the role of each component are explained in Versions of The Unicode Standard. See Unicode 13.0.0 for a summary of the contents of this version. That page also provides chapter-by-chapter links to the core specification and an index for block-by-block access to the code charts, for easier browsing of the content of the standard.
